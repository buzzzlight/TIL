# Values and Variables

## 변수 이름 짓기

- keyword는 변수로 쓸 수 없다 (function, new 등)

- 숫자가 맨 앞에 올 수 없다

- 대문자로 시작하지 않는다

- 낙타표기법? 사용

- _와 $만 사용가능한 기호

```js
// 좋은 예
let myFirstJob = 'Programmer';
let myCurrentJob = 'Teacher';

// 나쁜 예
let job1 = 'programmer';
let job2 = 'teacher';
```

## Data Types

### Object

```js
let me = {
  name: 'Subin'
};
```





### Primitive

``` js
let firstName = 'Subin';
let age = 27;
let fullAge = true;
let children;
```

- Number
  - Floating point numbers
  - Used for decimals and integer
  - 모든 숫자는 number 타입 (int와 float로 나뉘지 않음)

- String
  - Sequence of characters
  - Used for text
- Boolean
  - Logical type that can only be **true or false**
  - Used for taking decisions

- Undefined
  - Value taken by a variable that is not yet defined('empty value')
  - 값을 할당하지 않고 변수 선언만 한 상태 (빈 값)
- Null
  - Also means 'empty value'

- Symbol (ES2015)
  - Value that is unique and connot be changed
- BigInt (ES2020)
  - Larger integers than the Number type can hold



- 주석 ctrl + /



- dynamic typing

  - 쉽게 변수의 데이터 타입을 바꿀 수 있는 것

  ```js
  let javascriptIsFun = true;
  
  // 변수의 값을 변경할땐 let을 쓰지 않아도 됨
  // 처음 변수 선언할때만 let 씀
  javascriptIsFun = 'YES!';
  
  ```

  

const로 선언한 변수는 바뀌지않는값이다.. 한번 선언하면 바꿀수없따. 그리고 undefined로 선언할 수 없다

let > 나이같이 변할수 있는 변수

const > 출생년도와 같이 변할 수 없는 변수

그치만 되도록 const를 사용해 변수를 선언하는게 좋음
